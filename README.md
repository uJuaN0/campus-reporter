# NOVA Connect

NOVA Connect é uma plataforma digital desenvolvida durante a **Hackathon NOVA FCT** para melhorar a experiência no campus da **Universidade NOVA – Faculdade de Ciências e Tecnologia**.

Permite reportar problemas, gerir perdidos e achados, divulgar eventos e apoiar estudantes internacionais.

---

## Tecnologias Utilizadas

**Frontend**
- React
- TypeScript
- TailwindCSS

**Backend**
- Supabase

**Base de Dados**
- PostgreSQL (SQL)

**Outras funcionalidades**
- Autenticação com roles
- Sistema de permissões por utilizador
- Compressão automática de imagens
- Notificações e comentários
- Interface responsiva (mobile-friendly)

---

## Funcionalidades

- Reportar problemas no campus
- Sistema de perdidos e achados
- Gestão de eventos académicos
- Hub Internacional para estudantes estrangeiros
- Sistema de notificações por alteração de estado
- Painel de administração

---

## Tipos de Utilizador

**Administrador**
- gestão completa da plataforma
- criação e remoção de utilizadores
- atribuição de roles

**Departamentos**
- Problemas → gerir e atualizar estados
- Perdidos e Achados → gerir objetos
- Eventos → criar e editar eventos

**Alunos**
- visualizar conteúdos
- aceder ao Hub Internacional
- receber notificações
- não podem editar dados
